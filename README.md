# OpenAI Streaming API with FastAPI

## Tech stack

### server

- FastAPI
- LangChain
- uvicorn

### web

- Astro
- SolidJS
- TailwindCSS

## How to run

launch server

```bash
$ pip install -r requirements.txt
$ uvicorn main:app --port 8000 --reload
```

To view API docs in this url http://127.0.0.1:8000/docs

launch web

```bash
cd web
pnpm dev
```
